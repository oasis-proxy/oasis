# 数据管理与备份

Oasis 利用 Chrome 原生云基础设施进行安全、便携且同步的配置管理。

## 1. Chrome 云同步 (Chrome Cloud Sync)

**Chrome Sync Storage** 用于跨设备的设置自动同步，涵盖配置文件、策略和全局配置。

### 冲突解决

> [!IMPORTANT]
> 当本地与云端版本出现差异时，将显示“同步冲突”弹窗。

- **从云端拉取 (Pull from Cloud)**: 使用云端版本覆盖本地设置。
- **推送到云端 (Push to Cloud)**: 使用当前本地设置覆盖云端版本。
- **比较 (Compare)**: 提供修改时间戳、规则数量和配置文件数量的详细对比，辅助决策。

---

## 2. 备份与恢复

支持使用基于 JSON 的导入和导出工具进行手动迁移和离线备份。

### 完整配置导出

- 生成包含整个 Oasis 环境的单一 `.json` 文件。
- 包括代理服务器、代理组、PAC、自动策略和常规设置。

### JSON 导入

- **验证**: 自动验证导入的 JSON 文件以确保兼容性和安全性。
- **安全导入**: 导入过程将替换当前配置；建议在执行前导出当前设置。

### 数据维护

- **清除本地数据 (Clear Local)**: 将当前设备上的 Oasis 重置为默认状态。
- **清除同步数据 (Clear Sync)**: 删除 Chrome 云存储中的 Oasis 数据。
- _执行这些操作需要手动确认。_
