# Oasis - Smart Browser Proxy Manager

[English] | [**中文版**](README_zh.md)

Oasis is a modern, high-performance browser extension designed to give you complete control over your network traffic with advanced routing, monitoring, and synchronization features.

## Features

- **Protocol Support**: HTTP, HTTPS, SOCKS4, and SOCKS5.
- **Failover Chains**: Automated fallback logic for high availability.
- **Smart Rule Merging**: Intelligent optimization and contraction of routing rules.
- **Unsaved Changes Protection**: Strict navigation guards to prevent data loss during editing.
- **Context-Aware**: Add rules directly from the page, downloads, or monitoring logs.
- **Cross-Device Sync**: Synchronize your entire environment effortlessly via Chrome Cloud.
